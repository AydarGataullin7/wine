# Винный магазин

Код для отображения сайт с каталогом вин с данными из Excel файла.

## Установка и запуск

### 1. Клонирование репозитория
````bash
git clone <url-репозитория>
cd <папка-проекта>
````
### 2. Создание и активация виртуального окружения

````bash
python -m venv venv
````
````bash
venv\Scripts\activate
````
### 4. Установка зависимостей

````bash
pip install -r requirements.txt
````
### 5. Подготовка данных
Создайте файл ````assortment_of_wines.xlsx```` со следующей структурой:

- Категория - группа вин (обязательно)
- Название - название вина (обязательно)
- Сорт - сорт винограда (опционально)
- Цена - цена в рублях (обязательно, число)
- Картинка - имя файла изображения из папки images/ (обязательно)
- Акция - пометка "Выгодно" для товаров по акции (опционально)

Вы можете использовать файл ````wine.xlsx```` в папке проекта как образец. Скопируйте его и заполните своими данными.

### 6. Подготовка изображений 
Разместите изображения вин в папке images/ с именами, соответствующими столбцу "Картинка".

#### Файл assortment_of_wines.xlsx должен содержать столбцы в точности как указано в требованиях выше. Вы можете добавить дополнительные категории и вина по аналогии.

### 7. Запуск кода

````bash
python main.py
````
### В папке с проектом появится файл index.html. Откройте этот файл
#### Вы должны увидеть:
1. Главную страницу винного магазина
2. Карусель с изображениями
3. Разделы с винами, сгруппированные по категориям
4. Отзывы клиентов
5. Блок контактов

#### Остановка сервера:
Для остановки сервера нажмите Ctrl+C в терминале.

#### Деактивация виртуального окружения
После завершения работы:

````bash
deactivate
````
