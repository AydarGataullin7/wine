# Винный магазин

Код для отображения сайт с каталогом вин с данными из Excel файла.

## Установка и запуск

### 1. Клонирование репозитория
````
git clone <url-репозитория>
cd <папка-проекта>
````
### 2. Создание и активация виртуального окружения

````
python -m venv venv
````
````
venv\Scripts\activate
````
### 4. Установка зависимостей

````
pip install -r requirements.txt
````
### 6. Запуск кода

````
python main.py
````
### В папке с проектом появится файл index.html. Откройте этот файл
#### Вы должны увидеть:
1. Главную страницу винного магазина
2. Карусель с изображениями
3. Разделы с винами, сгруппированные по категориям
4. Отзывы клиентов
5. Блок контактов
